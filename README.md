# My Learning 🎓

A smart e-learning platform built using the MERN stack.

## 🚀 Features
- User authentication
- Course management
- Interactive learning system

## 🛠️ Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## 📁 Project Structure
- client/ → Frontend
- server/ → Backend

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   npm install
3. Run client and server
